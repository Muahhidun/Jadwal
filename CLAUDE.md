# Jadwal — исламское приложение напоминаний о поклонении

Этот файл автоматически читается Claude Code в начале каждой сессии. Он — точка входа для любого агента, продолжающего работу.

## Что это за проект
Мобильное приложение (Flutter, iOS + Android) — «ассистент поклонения»: напоминания, привязанные ко временам молитв (утренние/вечерние зикры, сура аль-Кахф и час дуа в пятницу), по тапу открываются сами зикры. Казахстан-first. Бесплатно, без рекламы, без бэкенда.

**Полный план, решения и архитектура: [PLAN.md](PLAN.md) — читать обязательно перед любой работой.**

## Жёсткие ограничения (нарушать нельзя)
1. **Никаких счётчиков-тасбихов** и цифровых счётчиков зикра — владелец считает это религиозным нововведением. Только отметка «выполнено» (галочка) на зикр целиком.
2. **Только достоверный религиозный контент** с указанием источника (сборник, номер хадиса) у каждого зикра. Религиозные тексты не придумывать и не «улучшать» — контент собирает и утверждает владелец.
3. **Никакой геймификации-самообмана**: без очков, бейджей, рейтингов. История/календарь постоянства — да.
4. Бесплатно, без рекламы, без сбора данных, всё хранится локально.

## Роли
- Владелец (не программист): продукт, контент, финальные решения.
- Claude Code: весь код, дизайн, архитектура. Объяснять технические вещи простым языком.
- Общение — на русском.
